# LXP B2B

AI-powered learning orchestration platform for Corporate L&D teams.

Initial product direction:

- B2B Corporate L&D first
- B2C expansion later through approved content, public previews, assessments, and certificate verification
- SME approval before learner publishing
- Source-grounded AI generation with prompt/version metadata

## Development

Install dependencies:

```bash
npm install
```

Run the app:

```bash
npm run dev
```

Run checks:

```bash
npm run typecheck
npm run lint
npm test
```
